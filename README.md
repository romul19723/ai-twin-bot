# AI-Двойник RoboForex — Telegram Bot

## Файлы проекта
- `roboforex_ai_twin_bot.py` — основной код бота
- `requirements.txt` — зависимости
- `Procfile` — команда запуска для Railway

## Быстрый запуск на Railway
1. Загрузи все 3 файла на GitHub в один репозиторий
2. Зайди на railway.app → New Project → Deploy from GitHub
3. Выбери репозиторий
4. Готово — бот запустится автоматически!
